# FITUR PROGRAM

di sini ada fitur CRUD dalam masing masing class, yaitu :
# ADD 
yang dimana ini bertujuan untuk menambahkan data baru mau itu buku, member ataupun peminjam

# READ 

yang dimana ini bertujuan untuk menunjukan data dan juga berhasil membaca data yang baru di tambahkan, di update ataupun di delete

# UPDATE

ini berfungsi untuk memperbarui data yang di mana data tersebut terdapat kesalahan ataupun kekurangan

# DELETE

ini berfungsi apabila data tersebut tidak ingin kita gunakan atau kita pakai dalam web kita


# ALUR
